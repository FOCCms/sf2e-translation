# Сообщить об ошибке

Если вы обнаружили ошибку, вы можете сообщить об этом двумя способами

### Телеграм

Написать об этом в [Telegram чат](https://t.me/sf2Eru/1) в тред с правками перевода.

### GitHub

Создать [тикет на гитхаб](https://github.com/FOCCms/sf2e-translation/issues). В этом случае ошибку будет проще отслеживать.

Чтобы это сделать
1. Перейдите по [ссылке](https://github.com/FOCCms/sf2e-translation/issues)
2. Авторизуйтесь
3. Нажмите "New issue"
4. Выберите подходящий шаблон
5. Опишите проблему
6. Нажмите "Create"
7. Готово!
