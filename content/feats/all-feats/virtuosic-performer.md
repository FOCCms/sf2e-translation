# Виртуозный исполнитель (Virtuosic Performer)

[[rules/traits/general|Общая]] [[rules/traits/skill|Навык]]

**Источник** Player Core  
**Тип** Способность  
**Уровень** 1  
**Предварительные условия** Тренирован Выступлению  

У вас выдающийся талант в одном виде выступления. Вы получаете бонус обстоятельства +1 при выполнении определённого вида выступления. Если вы мастер в Выступлении, этот бонус увеличивается до +2. Выберите одну из следующих специализаций и применяйте бонус при проверках Выступления этого вида. Если неясно, применяется ли специализация, решение принимает ГМ.

| Специализация           | Пример                                               |
| ----------------------- | ---------------------------------------------------- |
| Актерская игра (Acting) | Драма, прямые трансляции, реалити-шоу                |
| Комедия (Comedy)        | Буффонада, анекдоты, стендап                         |
| Танец (Dance)           | Балет, бальный, танец-история, поп, традиционный     |
| Клавишные (Keyboards)   | Клавитара, орган, фортепиано                         |
| Красноречие (Oratory)   | Эпос, ода, поэзия, рассказывание историй             |
| Перкуссия (Percussion)  | Колокольчики, бар чаймс, ударная установка, ксилофон |
| Пение (Singing)         | Баллада, напев, поп, рок, шанти                      |
| Струнные (Strings)      | Гитара, музыкальная катушка, терменвокс              |
| Синтезаторы (Synths)    | Синт дека, DJ-станции                                |
| Духовые (Winds)         | Горн, флейта, труба                                  |
