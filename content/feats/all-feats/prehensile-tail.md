# Цепкий хвост (Prehensile Tail)

[[rules/traits/ysoki|Исоки]]

**Источник** Player Core  
**Тип** Способность  
**Уровень** 5  

Ваш хвост функционирует как [[rules/player core/chapter-6-equipment#Многорукие персонажи (Multi-Armed Characters)|дополнительная конечность]]. Вы можете удерживать с его помощью количество снаряжения, эквивалентное одной руке, и можете использовать хвост для выполнения действий [[rules/activities/all-activities/interact|Взаимодействовать (Interact) ◆]]. Ваш хвост считается одной парой рук для целей обозначения активных рук при выполнении действия [[rules/activities/all-activities/switch-hands|Сменить Руки (Switch Hands) ◆]].
