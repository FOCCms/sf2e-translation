# Предусмотрительный расходник (Prescient Consumable)

[[rules/traits/general|Общая]]

**Источник** Player Core  
**Тип** Способность  
**Уровень** 7  
**Предварительные условия** [[feats/all-feats/prescient-planner|Дальновидный планировщик (Prescient Planner)]]  

Вы можете предсказать, какие расходуемые предметы могут понадобиться заранее. Используя [[feats/all-feats/prescient-planner|Дальновидный планировщик (Prescient Planner)]], вы можете достать из рюкзака расходуемый предмет вместо элемента снаряжения для приключений. Расходуемый предмет всё так же должен быть обычным, с уровнем не выше половины вашего уровня, и его Масса должна быть достаточно мала, чтобы его переноска не сделала вас [[rules/conditions/encumbered|перегруженным]].
