# Прицельное зрение (Scope Sight)

[[rules/traits/operative|Оперативник]]

**Источник** Player Core  
**Тип** Способность  
**Уровень** 1  

Наблюдать за меткой через прицел для вас так же естественно, как открыть глаза. Каждый раз, когда вы выполняете [[rules/activities/all-activities/aim|Прицелиться (Aim) ◆]], вы можете [[rules/activities/all-activities/interact|Взаимодействовать (Interact) ◆]], чтобы Активировать оптическую модификацию оружия.
