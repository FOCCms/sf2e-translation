# Трофейное ремесло (Monster Crafting)

[[rules/traits/general|Общая]] [[rules/traits/skill|Навык]]

**Источник** Player Core  
**Тип** Способность  
**Уровень** 7  
**Предварительные условия** Мастер Выживания  

Вы можете использовать части монстров при создании предметов. Вы можете использовать тело монстра для создания предметов, применяя Выживание вместо Ремесла для активности [[rules/activities/downtime/craft|Создать (Craft)]]. Если части тела монстра ценны, вы также можете потратить их на покрытие стоимости сырья для предмета. (ГМ принимает решение, если в описании монстра не указаны какие-либо ценные материалы из его тела.)

Если вы тратите дополнительные дни на работу над предметом, уменьшайте его Цену, используя уровень монстра вместо своего собственного. Определённые предметы могут требовать конкретных существ в качестве условия для активности [[rules/activities/downtime/craft|Создать (Craft)]].
