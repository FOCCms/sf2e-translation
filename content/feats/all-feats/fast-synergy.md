# Быстрая синергия (Fast Synergy)

[[rules/traits/envoy|Посланник]]

**Источник** Player Core  
**Тип** Способность  
**Уровень** 18  

Люди следуют за вами, потому что вы помогаете им раскрыть свой высший потенциал. Каждый из ваших союзников получает вторую реакцию, если начинает свой ход в пределах 100 футов от вас и может вас воспринимать. Эта вторая реакция может использоваться только для директив, которые вы отдаёте.
