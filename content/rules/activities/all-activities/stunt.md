# Выходка (Stunt) ◆

[[rules/traits/manipulate|Манипуляция]] [[rules/traits/move|Движение]] [[rules/traits/reckless|Безрассудство]]

**Источник** Player Core  
**Навык** Пилотирование  

Вы выполняете выходку во время Вождения транспорта, временно улучшая его эффективные возможности с риском потерять контроль. Ведите транспорт и выберите выходку. Все проверки Пилотирования, совершаемые как часть вашей Выходки, получают указанный штраф, включая проверки Пилотирования для выполнения [[rules/traits/reckless|безрассудного]] действия. Если и действие [[rules/activities/all-activities/drive|Вести (Drive) от ◆ до ◆◆◆]], и Выходка являются [[rules/traits/reckless|безрассудными]], вы должны совершить проверку Пилотирования для сохранения контроля над транспортом дважды.

| Выходка               | Штраф | Преимущество                                                            |
| --------------------- | ----- | ----------------------------------------------------------------------- |
| Отступление           | -1    | Двигаться назад на половину Скорости.                                   |
| Уклонение             | -1    | Двигаться на половину Скорости, чтобы получить бонус состояния +1 к КБ. |
| Разворот с ускорением | -1    | Двигаться на половину Скорости, затем повернуть в любом направлении.    |
| Бочка                 | -2    | Двигаться на половину Скорости, чтобы получить бонус состояния +2 к КБ. |
| Пролёт                | -2    | Двигаться на половину Скорости через клетку врага.                      |
| Дрифт                 | -2    | Двигаться на половину Скорости боком без поворота.                      |
| Разворот на месте     | -2    | Повернуть, затем двигаться на полную Скорость.                          |
