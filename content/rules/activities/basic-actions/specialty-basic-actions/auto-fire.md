# Автоматический огонь (Auto-Fire) ◆◆

[[rules/traits/area|Область]] [[rules/traits/attack|Атака]]

**Источник** Player Core  
**Требования** Вы держите оружие с признаком [[rules/traits/automatic|автоматическое]].  

Вы пытаетесь поразить каждое существо в [[rules/player core/chapter-8-playing-the-game#Конус (Cone)|конусе]] с дальностью, равной половине [[rules/player core/chapter-8-playing-the-game#Дистанция и досягаемость (Range and Reach)|шага дистанции]] оружия, без броска атаки. Любое существо в области получает урон оружием ([[rules/player core/chapter-8-playing-the-game#Простые спасброски (Basic Saving Throws)|простой спасбросок]] Рефлекса против КС вашего класса плюс значение [[rules/traits/tracking|наведение]] оружия). Это урон по области. Существа, которые критически проваливают этот спасбросок, подвергаются эффектам, происходящим при критическом попадании этим оружием, включая [[rules/player core/chapter-6-equipment#Эффекты критической специализации (Critical Specialization Effects)|эффект критической специализации]] оружия. Автоматический огонь имеет [[rules/player core/chapter-6-equipment#Расход (Expend)|расход]], равный числу целей в области × 2.
