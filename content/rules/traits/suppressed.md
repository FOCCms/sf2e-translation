# Подавление (Suppressed)

**Источник** Player Core

Существа, подавленные [[character-creation/all-classes-content/all-classes/soldier|Солдатом (Soldier)]], могут получать дополнительные состояния и другие негативные эффекты при применении способностей с признаком «подавление»; Солдат может применить только одну такую способность каждый раз, когда использует действие, наделённое признаком «подавление».
