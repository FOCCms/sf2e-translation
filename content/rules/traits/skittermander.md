# Скиттермандер (Skittermander)

**Источник** Player Core

Существо с этим признаком принадлежит к родословной скиттермандеров. Скиттермандеры по своей природе — импульсивные помощники. Умение с этим признаком может использоваться или выбираться только скиттермандерами.
