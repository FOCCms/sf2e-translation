# Дистанционный толчок (Ranged Shove)

**Источник** Player Core

Это оружие можно использовать, чтобы [[rules/activities/all-activities/shove|Толкнуть (Shove) ◆]] с помощью навыка Атлетика в пределах первого [[rules/player core/chapter-6-equipment#Дистанция (Range)|шага дистанции]] оружия. Проверка навыка получает штраф обстоятельства -2. Вы можете добавить бонус предмета к броскам атак как бонус к проверке. Оружие с признаком «Дистанционный толчок» не наносит урон при использовании, чтобы [[rules/activities/all-activities/shove|Толкнуть (Shove) ◆]].
