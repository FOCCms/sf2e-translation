# Универсальное (Versatile)

**Источник** Player Core

Универсальное оружие может использоваться для нанесения другого типа урона, помимо указанного. Этот признак указывает альтернативный тип урона. Например, колющее оружие с универсальным Р может наносить колющий или рубящий урон. Вы выбираете тип урона каждый раз, когда атакуете.
