# Архаичное (Archaic)

**Источник** Player Core

Доспехи с признаком архаичное изготовлены с использованием традиционных методов и материалов. Все доспехи из Pathfinder имеют признак архаичное. Руны доспехов функционируют нормально с архаичными доспехами.

Щит с признаком архаичное изготовлен с использованием традиционных методов и материалов, но не подходит для выдерживания атак современного оружия. Все щиты из Pathfinder имеют признак архаичное. Руны щитов функционируют нормально с архаичными щитами.

Оружие с признаком архаичное изготовлено с использованием традиционных методов и материалов. Всё оружие из Pathfinder имеет признак архаичное. Руны оружия функционируют нормально с архаичным оружием, но у этого оружия нет слотов для улучшений.
