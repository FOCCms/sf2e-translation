# Смертельное (Deadly)

**Источник** Player Core

При критическом попадании оружие добавляет к урону один куб урона оружия указанного размера. Этот бросок совершается после удвоения урона оружия. Количество кубов увеличивается до двух для оружия элитной и абсолютной градации и до трёх для оружия образцовой градации. Например, ==элитная наноклинковая рапира (elite nano-edge rapier)== наносит дополнительные 2d8 колющего урона при критическом попадании. Умение, изменяющее размер обычных кубов урона оружия, не изменяет размер «смертельного» куба.
