# Расходуемый (Consumable)

**Источник** Player Core

Предмет с этим признаком можно использовать только один раз. Если не указано иное, он уничтожается после активации. К расходуемым предметам относятся сыворотки и магические расходуемые предметы, такие как ампулы заклинаний и самоцветы заклинаний. Когда персонаж создаёт расходуемые предметы, он может изготавливать их партиями по четыре.
