# Громоздкий (Unwieldy)

**Источник** Player Core

Вы не можете использовать громоздкое оружие более одного раза за раунд для совершения [[rules/activities/basic-actions/strike|Удара (Strike) ◆]] и не можете использовать его для [[rules/activities/basic-actions/strike|Удара (Strike) ◆]] как часть реакции, например [[rules/activities/reactive-strike|Реакционный удар (Reactive Strike) ↩]].
