# Сообщить об ошибке

На текущий момент перевод насчитывает большое количество грамматических, логических и иных видов ошибок. Со временем перевод будет вычитываться и исправляться, но вы можете ускорить этот процесс, если найдёте ошибку и предложите вариант её исправления. Сделать это можно различными способами.
### Телеграм

Написать об этом в [Telegram чат](https://t.me/sf2Eru/1) в тред с правками перевода. Используйте этот вариант если на знакомы с GitHub или если ошибка достаточно мелкая и не требует обсуждений. Для более крупных ошибок рекомендуется механизм issue на GitHub.

### GitHub

Перевод позволяет создать [тикет на гитхаб](https://github.com/FOCCms/sf2e-translation/issues). В этом случае ошибку будет проще отслеживать. Также таким образом можно исключить сообщения об уже известных ошибках и детально обсудить каждую из них.

Чтобы это сделать
1. Перейдите по [ссылке](https://github.com/FOCCms/sf2e-translation/issues)
2. Авторизуйтесь
3. Нажмите "New issue"
4. Выберите подходящий шаблон
5. Опишите проблему
6. Нажмите "Create"
7. Готово!
