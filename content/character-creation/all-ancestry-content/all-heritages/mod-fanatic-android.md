# Андроид-фанатик модификаций (Mod Fanatic Android)

**Источник** Player Core  
**Тип** Наследие  

Ваше тело имеет один слот для [[rules/player core/chapter-6-equipment#Улучшения брони (Armor Upgrades)|улучшения брони]], который вы можете использовать для установки модификаций брони, и он не считается в общее количество улучшений вашей брони. Вы можете устанавливать и снимать модификации брони в свой личный слот улучшения, используя активность [[rules/activities/exploration/install-upgrade|Установить улучшение (Install Upgrade)]]. Выберите одно обычное улучшение брони с уровнем предмета 1 или ниже. Вы начинаете с этим улучшением брони, уже установленным в ваш слот улучшения (вам не нужно платить кредиты за покупку этого стартового улучшения).
