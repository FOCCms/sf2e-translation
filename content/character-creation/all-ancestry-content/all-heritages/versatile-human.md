# Универсальный человек (Versatile Human)

**Источник** Player Core  
**Тип** Наследие  

Люди были самой распространённой родословной на Голарионе до исчезновения этого мира и с тех пор распространились по звёздам благодаря своей универсальности. Выберите [[feats/general-feats|общую Способность]] по вашему выбору, для которой вы соответствуете предварительным условиям (как и Способность родословной, вы можете выбрать эту общую Способность на любом этапе [[rules/player core/chapter-1-introduction#Создание персонажа (Character creation)|создания персонажа]]).
