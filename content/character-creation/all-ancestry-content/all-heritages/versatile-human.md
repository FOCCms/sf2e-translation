# Разносторонний человек (Versatile Human)

**Источник** Player Core  
**Тип** Наследие  

Люди были самой распространённой родословной на Голарионе до исчезновения этого мира и с тех пор распространились по звёздам благодаря своей универсальности. Выберите [[feats/general-feats|общую способность]] по вашему выбору, для которой вы соответствуете предварительным условиям (как и способности родословной, вы можете выбрать эту общую способность на любом этапе [[rules/player core/chapter-1-introduction#Создание персонажа (Character creation)|создания персонажа]]).
