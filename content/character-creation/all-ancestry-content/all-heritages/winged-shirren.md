# Крылатый Ширрен (Winged Shirren)

**Источник** Player Core  
**Тип** Наследие  

Ваши предки были членами Роя, у которых развились крылья. У вас есть полностью функциональная пара крыльев, которая даёт вам [[rules/player core/chapter-8-playing-the-game#Скорость полёта (Fly Speed)|Скорость полёта]] 15 футов.
