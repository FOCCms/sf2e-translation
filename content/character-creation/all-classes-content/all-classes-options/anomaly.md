# Аномалия (Anomaly)

**Источник** Player Core  
**Тип** Парадокс исказителя

### Квантовое поле (Quantum Field)

Движения размываются, пока образы альтернативных версий ваших врагов мерцают вокруг них. Когда враг в вашем квантовом поле пытается совершить действие с признаком [[rules/traits/concentrate|концентрация]], действие прерывается, если только он не преуспеет в [[rules/player core/chapter-8-playing-the-game#Чистые проверки (Flat Checks)|чистой проверке]] с КС 5. Первый раз, когда существо проваливает эту проверку, оно получает временный иммунитет на 24 часа.
