# С грохотом орудий (Guns Blazing)

**Источник** Player Core  
**Тип** Лидерский стиль  

Вы ведёте свою команду, производя впечатление огневой мощью. Ваша громогласная личность подкреплена градом выстрелов, и вы всегда стреляете первым.

**Лидерский навык** [[character-creation/skills/acrobatics|Акробатика]]  
**Лидерский перк** Вы получаете [[feats/all-feats/incredible-initiative|Невероятная инициатива (Incredible Initiative)]] как бонусную способность.
