# Солнечное оружие (Solar Weapon)

**Источник** Player Core  
**Тип** Солнечное проявление солариана  

Силой одной лишь воли вы вызываете осязаемое оружие, выкованное из сущности звёзд. Это оружие является продолжением вас самих, и вы можете изменять его форму с помощью медитации. При проявлении вашего оружия оно появляется в свободной руке по вашему выбору. Если у него есть признак [[rules/traits/free-hand|свободная рука]], оно может проявляться в любой руке. Ваше солнечное оружие является [[rules/player core/chapter-6-equipment#Категории оружия (Weapon Categories)|воинским]] оружием ближнего боя. Оно наносит 1d8 дробящего, колющего или рубящего урона на ваш выбор с признаками [[rules/traits/attuned|сонастроен]] и [[rules/traits/solarian|солариан]].

На первом уровне выберите до двух из следующих признаков для вашего оружия. В качестве альтернативы вы можете выбрать признак [[rules/traits/reach|досягаемость]] или [[rules/traits/two-hand|двуручное]] 1d10 для вашего оружия вместо выбора двух следующих признаков.

- [[rules/traits/disarm|Разоружение]]
- [[rules/traits/free-hand|Свободная рука]]
- [[rules/traits/grapple|Захват]]
- [[rules/traits/nonlethal|Несмертельное]]
- [[rules/traits/parry|Парирование]]
- [[rules/traits/shove|Толчок]]
- [[rules/traits/sweep|Размах]]
- [[rules/traits/trip|Опрокидывание]]
- [[rules/traits/versatile|Универсальное]] (только Д/К/Р)
- [[rules/traits/reach|Досягаемость]] (считается за 2 признака)
- [[rules/traits/two-hand|двуручное]] (считается за 2 признака)

Вы можете выбрать одну из следующих [[rules/player core/chapter-6-equipment#Группа (Group (weapon))|групп]] оружия, частью которой будет ваше оружие.

- [[equipment/all-equipment/all-specialization-effects/axe|Топор]]
- [[equipment/all-equipment/all-specialization-effects/brawling|Драка]]
- [[equipment/all-equipment/all-specialization-effects/club|Дубинка]]
- [[equipment/all-equipment/all-specialization-effects/flail|Цеп]]
- [[equipment/all-equipment/all-specialization-effects/hammer|Молот]]
- [[equipment/all-equipment/all-specialization-effects/knife|Нож]]
- [Клевец](https://pf2e-ru-translation.readthedocs.io/ru/latest/equipment/Weapons.html#weapon-crit-spec-pick)
- [[equipment/all-equipment/all-specialization-effects/polearm|Древковое]]
- [[equipment/all-equipment/all-specialization-effects/spear|Копье]]
- [[equipment/all-equipment/all-specialization-effects/sword|Меч]]

Ваше солнечное оружие получает дополнительные преимущества в зависимости от вашей текущей сонастройки.

### Гравитон-сонастроен

Когда вы успешно совершаете [[rules/activities/all-activities/strike|Удар (Strike) ◆]] по существу, используя ваше солнечное оружие, пространство этого существа и все смежные с ним клетки становятся для этого существа [[rules/player core/chapter-8-playing-the-game#Сложная местность (Difficult Terrain)|сложной местностью]] до конца его следующего хода.

### Фотон-сонастроен

Когда вы успешно совершаете [[rules/activities/all-activities/strike|Удар (Strike) ◆]] по существу вашим солнечным оружием, вы наносите дополнительное количество урона огнём, равное половине вашего уровня (округляя вверх). Ваше солнечное оружие излучает яркий свет в радиусе 5 футов (и [[rules/player core/chapter-8-playing-the-game#Тусклый свет (Dim Light)|тусклый свет]] на следующие 5 футов).
