# Артиллерист (Bombard)

**Источник** Player Core  
**Тип** Боевой стиль солдата  

Нет ничего лучше большой пушки (а может, и нескольких), чтобы пережить тяжёлые времена. Вы смирились с тем, что случайная пуля из одного из ваших залпов иногда может задеть союзника, но стараетесь свести такие случаи «дружественного огня» к минимуму. Вы натренировались, чтобы избегать вреда для союзников при своих залпах, и научились мастерски направлять атаки вокруг них.

Когда вы атакуете оружием по области, вы можете выбрать количество союзников в области, равное половине вашего модификатора Телосложения. Выбранные союзники не затрагиваются атакой. Кроме того, враги, которые преуспели (но не получили критический успех) при спасброске против вашей атаки по области, всё равно становятся [[rules/conditions/suppressed|подавлены]] до начала вашего следующего хода.
