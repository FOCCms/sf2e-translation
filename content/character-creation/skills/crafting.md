# Ремесло (Crafting)
*Интеллект*

Вы можете использовать этот навык для создания и ремонта предметов.

## Действия Ремесла

### Нетренированные действия Ремесла (Crafting Untrained Actions)

Даже если вы нетренированы Ремеслу, вы можете использовать его для следующих общих действий навыка.

- **[[rules/activities/recall-knowledge|Вспомнить информацию (Recall Knowledge) ◆]]** о ценности предметов, техническом снаряжении, инженерии, необычных материалах и технологических существ.

![[rules/activities/exploration/repair#Ремонт (Repair)]]

### Тренированные действия Ремесла (Crafting Trained Actions)

Вы должны быть тренированы Ремеслу, чтобы использовать следующие общие действия навыков.

- [[rules/activities/downtime/earn-income|Заработок денег (Earn Income)]] производя товары для рынка.
