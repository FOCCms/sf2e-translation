# Чип данных (Data Chip)

[[rules/traits/tech|Техно]]

**Источник** Player Core  
**Тип** Предмет  
**Уровень** 0  
**Цена** 2 кред.  
**Масса** —  

Чип данных содержит достаточно памяти для религиозного текста, фотоальбома, личного журнала или подобной информации. Вы можете просматривать данные на чипе с помощью компьютера, например встроенного в ваш [[equipment/all-equipment/all-items/comm-unit|коммуникатор]].
