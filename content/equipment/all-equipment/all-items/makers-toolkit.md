# Комплект ремесленника (Maker's Toolkit)

[[rules/traits/tech|Техно]]

**Источник** Player Core  
**Тип** Предмет  

Для создания предметов из [[rules/player core/chapter-6-equipment#Универсальная полимерная основа (УПО) (Universal Polymer Base, UPB)|УПО]] с помощью навыка Ремесло вам необходим комплект ремесленника.

## Комплект ремесленника (коммерческий)

**Источник** Player Core  
**Тип** Предмет  
**Уровень** 0  
**Цена** 40 кред.  
**Руки** 2  
**Масса** 2  

## Комплект ремесленника (тактический)

**Источник** Player Core  
**Тип** Предмет  
**Уровень** 3  
**Цена** 500 кред.  
**Руки** 2  
**Масса** 2  

Тактический комплект ремесленника даёт бонус предмета +1 к проверке.
