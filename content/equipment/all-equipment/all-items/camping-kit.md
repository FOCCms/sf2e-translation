# Набор для кемпинга (Camping Kit)

**Источник** Player Core  
**Тип** Предмет  
**Уровень** 0  
**Цена** 30 кред.  
**Руки** 2  
**Масса** 2  

Набор для кемпинга содержит всепогодную палатку для четырёх существ Среднего размера или двух существ Большого размера, спальные мешки, разжигатель огня, недельный запас пайков и нож выживания.
