# Инвалидное кресло (Wheelchair)

[[rules/traits/tech|Техно]]

**Источник** Player Core  
**Тип** Предмет  
**Уровень** 0  
**Цена** 1 кред.  
**Масса** 2  

Инвалидные кресла бывают самых разных размеров и стилей, подходящих для множества инопланетных типов тел, включая парящие кресла. Автоматизированные механизмы позволяют инвалидным креслам преодолевать ступеньки, лестницы и [[rules/player core/chapter-8-playing-the-game#Неровная земля (Uneven Ground)|неровную землю]] без дополнительных затруднений ([[rules/player core/chapter-8-playing-the-game#Сложная местность (Difficult Terrain)|сложная местность]] по-прежнему влияет на вашу Скорость при использовании инвалидного кресла).
