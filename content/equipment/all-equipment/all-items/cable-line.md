# Трос (Cable Line)

**Источник** Player Core  
**Тип** Предмет  

Этот промышленный трос обычно изготавливается из прочного пластика, покрытого металлом. Коммерческий трос покрыт титановым сплавом.

## Трос (коммерческий)

**Источник** Player Core  
**Тип** Предмет  
**Уровень** 0  
**Цена** 2 кред.  
**Масса** Л  

## Трос (тактический)
