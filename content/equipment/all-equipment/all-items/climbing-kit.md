# Набор для скалолазания (Climbing Kit)

**Источник** Player Core  
**Тип** Предмет  

Коммерческий набор для скалолазания включает 50 футов троса, крюк-кошку, страховочные ремни, блоки, крюки, молоток и кошки. Наборы для скалолазания позволяют закрепиться на стене, по которой вы пытаетесь [[rules/activities/all-activities/climb|Карабкаться (Climb) ◆]], передвигаясь вдвое медленнее обычного (минимум 5 футов), но позволяя совершить чистую проверку КС 5 при критическом провале, чтобы предотвратить [[rules/player core/chapter-8-playing-the-game#Падение (Falling)|падение]]. Один набор содержит достаточно материалов только для одного скалолаза; каждому скалолазу нужен собственный набор.

## Набор для скалолазания (коммерческий)

**Источник** Player Core  
**Тип** Предмет  
**Уровень** 0  
**Цена** 5 кред.  
**Руки** 2  
**Масса** 1  

## Набор для скалолазания (тактический)

**Источник** Player Core  
**Тип** Предмет  
**Уровень** 3  
**Цена** 40 кред.  
**Руки** 2  
**Масса** 1  

Вы получаете бонус предмета +1 к проверкам Атлетики, чтобы [[rules/activities/all-activities/climb|Карабкаться (Climb) ◆]] при помощи тактического набора для скалолазания.
