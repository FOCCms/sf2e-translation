# Гигиенический набор (Hygiene Kit)

**Источник** Player Core  
**Тип** Предмет  
**Уровень** 0  
**Цена** 2 кред.  
**Руки** 2  
**Масса** Л  

Этот набор содержит все инструменты и средства, необходимые для ухода за собой и поддержания гигиены. Наборы, разработанные для определённых существ, включают принадлежности (например, средства по уходу за чешуёй для весков или масло для чистки усиков для лашунт и ширренов), которые менее полезны для других существ.
