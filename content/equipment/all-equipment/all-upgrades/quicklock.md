# Быстрозахват (Quicklock)

[[rules/traits/uncommon|Необычный]] [[rules/traits/magical|Магический]] [[rules/traits/tech|Техно]]

**Источник** Player Core  
**Тип** Улучшение  
**Уровень** 16  
**Цена** 100000  
**Использование** устанавливается в оружие  
**Масса** Л  

Оружие с быстрозахватом оснащено несколькими устройствами слежения и дополнительными компьютерными процессорами для ускорения встроенных систем наведения, позволяя владельцу атаковать быстрее. Пока вы держите оружие с быстрозахватом, вы получаете состояние [[rules/conditions/quickened|ускорен]], но можете использовать предоставленное дополнительное действие только для совершения [[rules/activities/all-activities/strike|Удара (Strike) ◆]] этим оружием.
