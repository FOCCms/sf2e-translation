# Эмодермис (Moodskin)

[[rules/traits/magical|Магический]] [[rules/traits/magitech|Магитех]]

**Источник** Player Core  
**Тип** Аугментация  
**Уровень** 0  
**Цена** 35  

Магически насыщенные наниты, внедрённые в вашу кожу, автоматически реагируют на ваше настроение, изменяя её пигментацию. Изменение цвета происходит постепенно, обычно занимая не менее 30 секунд для полной смены одного оттенка на другой. Эмодермис может быть настроен при установке, чтобы определять, какие цвета соответствуют каждому настроению. Эта аугментация не учитывается в вашем [[rules/player core/chapter-6-equipment#Лимит имплантов (Implant Limit)|лимите имплантов]].
