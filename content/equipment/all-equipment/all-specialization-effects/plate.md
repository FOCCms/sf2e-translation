# Латная (Plate)

**Источник** Player Core  
**Тип** Эффект специализации брони  

Прочная пластина не даёт опоры для режущего лезвия. Вы получаете [[rules/player core/chapter-8-playing-the-game#Сопротивление (Resistance)|сопротивление]] рубящему урону, равное 1 + значение [[rules/traits/resilient|стойкий]] брони для средней брони или 2 + значение [[rules/traits/resilient|стойкий]] брони для тяжёлой брони.
