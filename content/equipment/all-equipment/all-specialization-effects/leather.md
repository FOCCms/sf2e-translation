# Кожаная (Leather)

**Источник** Player Core  
**Тип** Эффект специализации брони  

Толстый второй слой этой брони рассеивает ударную силу, уменьшая дробящий урон. Вы получаете [[rules/player core/chapter-8-playing-the-game#Сопротивление (Resistance)|сопротивление]] дробящему урону, равное 1 + значение [[rules/traits/resilient|стойкий]] брони для средней брони или 2 + значение [[rules/traits/resilient|стойкий]] брони для тяжёлой брони.
